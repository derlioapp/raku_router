The example app below runs as a real Flutter web build. Navigate the tabs, open a
note, push a full-page photo — and watch the **address bar change** and browser
**back/forward** work, with each tab keeping its own history.

</div>

:::note[Building the demo]
The demo is the package's `example/` app, built for web and served from
`/raku_router/demo/` (the site's GitHub Pages base). To (re)generate it:

```bash
cd example
flutter build web --base-href /raku_router/demo/
cp -R build/web/* ../website/public/demo/
```

For runnable *snippets* (rather than the whole app), embed
[DartPad](https://dartpad.dev) iframes the way the Flutter docs do.
:::
